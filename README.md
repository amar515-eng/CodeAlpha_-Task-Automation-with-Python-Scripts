## Project Overview

This project is a simple and efficient Python automation script developed to extract all valid email addresses from a text file and save them into another file. The application uses Regular Expressions (Regex) to identify email addresses automatically, eliminating the need for manual searching.

The program reads the contents of an input text file, extracts every valid email address, displays the extracted emails on the screen, and stores them in a separate output file named **emails.txt**. This project demonstrates how Python can automate repetitive text-processing tasks quickly and accurately.
